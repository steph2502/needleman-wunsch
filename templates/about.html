About Needleman–Wunsch

The Needleman–Wunsch algorithm performs global sequence alignment by using dynamic programming to find the optimal end-to-end alignment between two sequences.

Core idea

Build a scoring matrix \(F\) where each cell \(F[i][j]\) stores the best score for aligning the first \(i\) characters of sequence 1 to the first \(j\) characters of sequence 2. Then perform a traceback from the bottom-right cell to recover an optimal alignment.

Scoring model

  • Match: add match score
  • Mismatch: add mismatch penalty (usually negative)
  • Gap: add gap penalty (usually negative)

Matrix recurrence

F[i][j] = max(
F[i-1][j-1] + s(xᵢ, yⱼ),
F[i-1][j] + gap,
F[i][j-1] + gap
)
